Research + Hardware

Outdoor mmWave channel sounding at 28 GHz

Built and deployed a coherent distributed MIMO channel sounder integrating GPSDO-disciplined synchronization, IF-over-fiber interconnects, phased-array systems, and outdoor receiver/transmitter carts for urban 28 GHz propagation measurements.

Hardware stack
GPS clocks, phased arrays, IF-over-fiber links, battery-powered mobile carts, RF synchronization, outdoor measurement rigs.
Research focus
Distributed MIMO sounding, synchronization stability, wideband propagation characterization, and 6G urban channel measurements.

Signal Processing

Wideband PDP multipath component extraction

Designed a custom multi-stage MPC extraction pipeline in MATLAB for wideband PDP channel data without angular information, treating the delay-power profile as a 2D image for robust peak detection followed by DBSCAN clustering with geometric target organization.

Pipeline
Peak detection, stitched trajectory tracking, and DBSCAN clustering of delay-power paths.
Output
Per-MPC delay profiles recovered via constrained least squares with differential matrix formulation.

Communications

Pilot-coded OTFS channel estimation

Implemented embedded pilot code-aided delay-Doppler estimation for OTFS and extended the idea to MIMO using code sets with strong correlation properties.

Key result
Reduced PAPR while preserving accurate channel estimation and BER performance compared to a single-symbol pilot.
Paper outcome
IEEE CCNC 2026 publication on embedded pilot code-aided delay-Doppler channel estimation.

Optical Systems

Low-complexity equalization techniques for multi-mode fibers

Built a multi-section MMF channel model in the strong-coupling regime and studied modal dispersion, mode-dependent losses, mutual information, and achievable rates under varying fiber parameters.

Model
Strong coupling, multi-section propagation, GD and MDL analysis, and MMSE/LMMSE-style receiver behavior.
Results
Capacity and penalty trends versus mode count, section length, and amplifier spacing.

Machine Learning + RF

Drone RF Signal Classification Using Feature Engineering and NumPy-Based Models

Developed lightweight RF drone signal classification pipelines using 89 engineered IQ-domain features spanning spectral, temporal, phase, and burst statistics. Evaluated hierarchical SVMs, MLPs, and ensemble classifiers on 17,000+ RF captures across seven drone/noise classes.

Feature Engineering
Spectral entropy, PAPR, kurtosis, band-energy ratios, temporal burst statistics, and IQ-domain features extracted from raw RF captures.
Best Result
Ensemble of hierarchical SVM and MLP models achieved 66.9% balanced classification accuracy across seven RF classes.
