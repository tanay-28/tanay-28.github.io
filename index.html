DARPA · Hardware · RF Systems

Outdoor mmWave channel sounding at 28 GHz

Sole engineer on a DARPA-funded coherent distributed MIMO channel sounder. Designed and integrated the complete measurement cart — ZCU670 RFSoC TX/RX front end, GPSDO timing reference, distribution amplifier, signal generator with differential clock, output BPF, RF power amplifier, and antenna front end. Deployed across three transmitter carts and a receiver cart for outdoor urban mmWave propagation measurements.

ZCU670 RFSoC Simulink + Vivado GPSDO IF-over-fiber 28 GHz
Outcome
Outdoor urban mmWave measurement campaign complete. Journal paper in preparation.

Signal Processing · MATLAB

Wideband PDP multipath component extraction

Designed a custom multi-stage MPC extraction pipeline for wideband PDP channel data without angular information. Treats the delay-power profile as a 2D image for robust peak detection, followed by DBSCAN clustering. Per-MPC delay profiles recovered via constrained least-squares with a differential matrix formulation.

MATLAB DBSCAN Least squares 3–4 GHz
Outcome
Enabling cell-free massive MIMO channel path correlation analysis from scalar PDP data alone.

Communications · Published

Pilot-coded OTFS delay-Doppler channel estimation

Proposed embedding a Barker code pilot sequence in the OTFS delay-Doppler transmit grid instead of a single amplified pilot symbol. Cross-correlation at the receiver achieves accurate tap detection without PAPR penalty — achieving ~6 dB PAPR reduction while matching ideal channel estimation BER performance.

OTFS Channel estimation Barker code PAPR reduction MATLAB
Outcome
Published — IEEE CCNC 2026, Las Vegas. DOI: 10.1109/CCNC65079.2026.11366337

Communications · Published

OTFS for underwater acoustic communications

Developed an OTFS-based underwater acoustic communication system in MATLAB. Modeled medium-depth channel scenarios in both delay-time and delay-Doppler domains. Achieved BER = 10⁻³ at 25 dB SNR for a 10 kHz acoustic signal at 100 m depth using an MPA detector.

OTFS Underwater acoustics MPA detector MATLAB
Outcome
Published — APWiMob 2025.

Machine Learning · RF

Drone RF signal classification

Engineered an 89-feature IQ signal extraction pipeline — spectral entropy, PAPR, kurtosis, band-energy ratios, temporal burst statistics — from 17,000+ raw RF captures across seven drone and noise classes. Hierarchical SVM and MLP ensemble achieved 66.9% balanced accuracy across −12 to +30 dB SNR. SNR-stratified analysis identified scalar feature information loss as the performance bottleneck, motivating a CNN extension.

IQ signals SVM MLP Feature engineering Python
Next step
CNN extension in PyTorch — replacing scalar feature pipeline with end-to-end deep learning on raw IQ.

Optical Systems · TU Munich

Linear channel modeling for multi-mode fibers

Bachelor's thesis at TU Munich under Prof. Norbert Hanik. Modeled the MMF channel using a multi-section matrix propagation framework in the strong coupling regime. Analyzed modal dispersion and mode-dependent losses. Implemented ZF and LMMSE equalizers and computed MMSE achievable information rates across fiber parameter configurations via Monte Carlo simulation.

MMF LMMSE Mode coupling MATLAB MDM
Outcome
Undergraduate thesis, TU Munich Institute for Communications Engineering, Dec. 2024.
